# Prompt Design Framework for Quality Engineering

## RACE Framework

A reusable framework for creating testing prompts.

### R - Role

Define the AI role.

Example:

```text
Act as a Senior Test Architect.
```

---

### A - Action

Specify required task.

Example:

```text
Generate test cases.
```

---

### C - Context

Provide business context.

Example:

```text
Online Banking Application
```

---

### E - Expected Output

Specify format.

Example:

```text
Provide output in tabular format.
```

---

## Template

```text
Role:
Action:
Context:
Expected Output:
Constraints:
```
