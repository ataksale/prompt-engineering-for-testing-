# Performance Testing Prompts

## Objective

Use AI to identify performance risks, create workload models, and generate performance testing strategies.

---

```text
Act as a Performance Test Architect.

Create a performance testing strategy for the following application.

Include:

- Load Testing
- Stress Testing
- Spike Testing
- Endurance Testing
- Scalability Testing

Application Details:

[Paste Application Description]
```

