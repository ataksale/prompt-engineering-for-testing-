# Test Case Generation Prompts

## Objective

Generate comprehensive test cases from requirements, user stories, and acceptance criteria.

---

## Prompt 1: Functional Test Case Generation

### Prompt

```text
Act as a Senior Test Architect.

Analyze the following requirement and generate comprehensive test cases.

Include:

- Test Case ID
- Test Objective
- Preconditions
- Test Steps
- Expected Results
- Priority
- Positive Scenarios
- Negative Scenarios
- Boundary Conditions

Requirement:

[Paste Requirement Here]
```

### Expected Output

* Functional test cases
* Negative test cases
* Boundary value scenarios
* Risk-based coverage

---

## Prompt 2: User Story Based Testing

```text
Act as an ISTQB-certified Test Lead.

Generate exhaustive test scenarios for the following user story.

Include:

- Functional Tests
- Usability Tests
- Security Tests
- Error Handling Tests
- Edge Cases

User Story:

[Paste User Story]
```

---

## Prompt 3: Risk-Based Testing

```text
Identify high-risk business scenarios for the following feature.

Rank them as:

- Critical
- High
- Medium
- Low

Generate corresponding test cases.
```
