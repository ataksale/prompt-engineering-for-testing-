# Root Cause Analysis Prompts

## Prompt 1: Five Whys Analysis

```text
Act as a Software Quality Consultant.

Perform a Five Whys Analysis for the following defect.

Provide:

1. Problem Statement
2. Why #1
3. Why #2
4. Why #3
5. Why #4
6. Why #5
7. Root Cause
8. Corrective Actions
```
