# Test Data Generation Prompts

## Prompt 1: Synthetic Customer Data

```text
Generate realistic synthetic customer data.

Requirements:

- 100 Records
- UK Customers
- GDPR Compliant

Fields:

- Name
- Address
- Email
- Phone Number
```

## Prompt 2: Payments Test Data

```text
Generate test data for payment processing.

Provide positive and negative test data.
```
