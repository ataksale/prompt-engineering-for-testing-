## Overview

Generative AI is transforming Software Quality Engineering by enabling testers, developers, architects, and product teams to automate complex activities using natural language prompts.

This repository provides a curated collection of production-ready prompts for:

* Test Case Generation
* Test Scenario Design
* Defect Analysis
* Root Cause Analysis
* Test Data Generation
* API Testing
* Performance Testing
* Security Testing
* Test Automation
* Requirements Analysis

The goal is to help Quality Engineering teams leverage AI effectively and consistently across the Software Development Lifecycle (SDLC).
